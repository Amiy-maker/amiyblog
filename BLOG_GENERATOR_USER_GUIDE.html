Understanding Section Markers

What Are Section Markers?

Section markers are special tags that divide your content into meaningful sections. Each section type has specific HTML formatting rules optimized for SEO and readability.

‚ÑπÔ∏è Format: Section markers use the format {sectionX} where X is a number from 1-12

The 12 Section Types

Your blog post is divided into 12 predefined section types, each serving a specific purpose and optimized for search engines:

Section Name Purpose
{section1} Hero/Title Main headline and introduction
{section2} Intro Paragraph Opening paragraph setting context
{section3} Table of Contents Overview of what readers will learn
{section4} Key Benefits/Overview Main benefits or overview of topic
{section5} Section Body Main content sections
{section6} Statistics/Facts Data, statistics, factual information
{section7} Comparison Comparisons between options
{section8} Expert Quote Quotes from experts or authorities
{section9} How-To Steps Step-by-step instructions
{section10} Internal Links Links to relevant articles
{section11} FAQs Frequently Asked Questions
{section12} CTA/Conclusion Call-to-Action and closing remarks

Method 1: Using Quick Insert Buttons

The easiest way to insert section markers is using the Quick Insert buttons in the left sidebar.

üì∏ Quick Insert Buttons
Shows all 12 quick insert buttons organized in the sidebar

To Insert a Section:

  1. Click where you want to insert the marker in the editor
  2. Click the corresponding button (e.g., "section1 Hero/Title")
  3. The marker will be automatically inserted at your cursor position

Method 2: Manual Typing

You can also manually type section markers directly in the editor. Simply type {section1}, {section2}, etc., wherever you want to mark sections.

Tips & Best Practices

Content Organization

  • Always start with section1 - Hero/Title is your main heading
  • Follow with section2 - Set up your topic with context
  • Use section3 - Help readers navigate with a table of contents
  • Fill sections 4-11 - Your main content and supporting information
  • End with section12 - Strong closing with clear call-to-action

SEO Optimization

  • Use Descriptive Titles: Include keywords naturally in section headings
  • Write for Readers First: Focus on providing genuine value
  • Include Statistics: Add credibility with data and research
  • Add Expert Quotes: Reference authorities in your field
  • Use Internal Links: Link to other relevant content on your site
  • Add Clear Instructions: Step-by-step information improves user experience

Image Best Practices

  • Use High Quality: Minimum 1200px width for clarity
  • Compress Images: Reduce file size for faster loading
  • Descriptive Names: Use keyword-rich filenames for SEO
  • Add Context: Include alt text descriptions in your content

Content Length Recommendations

  • Total Length: Aim for 1,500-2,500 words
  • Vary Section Length: Mix short and detailed sections
  • Short Sections: Use for steps, FAQs, and lists (100-300 words)
  • Longer Sections: Use for detailed explanations (500-800 words)

Publishing Strategy

  • Consistency Beats Perfection: Regular publishing builds audience
  • Ideal Frequency: 2-4 posts per month
  • Follow a Schedule: Plan your publishing dates in advance
  • Update Old Posts: Refresh older posts with new information quarterly

Troubleshooting

Common Issues & Solutions

‚ùå "Unsupported file format"

Solution: Ensure your file is .txt, .md, or .docx. Rename the file extension if necessary. Common issue: uploading .doc instead of .docx for Word files.

‚ùå "Please write some content first"

Solution: Add content to the editor before clicking Generate. Include at least one section marker {section1}, {section2}, etc.

‚ùå "Missing Required Sections"

Solution: Review the warning list and add the missing section types. Your blog needs essential sections like introduction and conclusion.

‚ùå "Images not uploading"

Solutions:

  • Check image file format (must be JPG, PNG, WebP, or GIF)
  • Ensure image file size isn't too large (under 5MB recommended)
  • Check your internet connection
  • Try uploading a different image to test

‚ùå "Failed to publish to Shopify"

Solutions:

  • Verify your Shopify connection is configured correctly
  • Check that all required fields (Blog Title) are filled
  • Try again in a few moments
  • Check that you have write permissions for blogs

‚ùå "HTML looks different in browser"

Solution: The preview tab shows raw HTML code. Open the downloaded .html file in your web browser to see the formatted result.
